# 🛒 Ecomarce - MERN Stack E-commerce Website

**Ecomarce** is a full-stack e-commerce website built using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application supports user and admin roles, with full product, cart, order, and payment functionality.

---



## 📌 Key Features

### 👤 User Side:
- Register / Login / Logout
- Browse products with search and filters
- Product details view
- Add to cart and manage cart items
- Place orders with address and payment option (COD, eSewa, Khalti)
- View order history and profile

### 🛠️ Admin Panel:
- Admin authentication
- Create, update, delete products
- Manage all orders
- Manage users
- Admin dashboard for stats and control

---

## 🛠️ Tech Stack

| Frontend         | Backend         | Database | Styling        |
|------------------|------------------|----------|----------------|
| React.js (with Redux) | Node.js + Express.js | MongoDB | Bootstrap CSS or Inline JS |
